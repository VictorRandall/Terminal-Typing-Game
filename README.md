# Terminal-Typing-Game
## "Imitation is the Sincerest Form of Flattery"

This is a game about learning source code. You can use any repo as it only needs text files. 

Its purpose is to help understand the code by making it yourself.

![Preview](/Screenshot.png)

## Usage

```bash
cargo run <file1> <file2> <etc>
```

## Linux

If you are on linux, you should run the following installations:

```bash
sudo apt install libasound2-dev
```
